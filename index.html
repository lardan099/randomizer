Что посмотреть?

Нужен бесплатный API-ключ от TMDB.

1. Зайди на themoviedb.org/signup и зарегистрируйся
2. После входа → Settings → API → Request an API Key
3. Выбери «Developer», заполни форму (можно писать что угодно)
4. Скопируй API Key (v3 auth) и вставь сюда
